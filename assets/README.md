# assets/

Placeholder home for art, audio, and other imported resources.

Right now the prototype draws **everything in code** (`_draw()` with simple
shapes) — there are no image or audio files to import, so this folder is
intentionally near-empty. That's a deliberate Phase 1 choice: game feel first,
no asset pipeline to go stale, nothing that can't be reproduced from a clean
clone.

When real assets arrive (Phase 2+), keep them here, grouped by type:

```
assets/
├── sprites/
├── audio/
└── fonts/
```

Keep assets **reproducible and openly licensed** — no private or
non-redistributable files in a public repo.
